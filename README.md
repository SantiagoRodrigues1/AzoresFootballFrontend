# AzoresScore - Live Football Application

A modern Ionic + React + TypeScript application for tracking Azores football matches, teams, and player management with **Team Manager** role support.

## 🎯 Key Features

### For Fans
- View upcoming matches and standings
- Browse teams and player profiles
- Live match updates
- Statistics and rankings

### For Team Managers 🆕
- **Match Call-Ups**: Select 19-player squads (11 starters + 8 bench) 24 hours before match
- **Formation Selection**: Choose from 5 professional formations (4-3-3, 4-4-2, 3-5-2, 4-2-3-1, 5-3-2)
- **Visual Lineup Builder**: Interactive football pitch for building starting XI
- **Time-Based Permissions**: Automatic unlock windows (24h and 1h before match)
- **Real-time Countdown**: See exactly when you can take actions

### For Admins
- Full match, team, and player management
- Admin dashboard with audit logs
- User and referee management
