3 CAMADAS DE PROTEÇÃO:

1. FRONTEND (AdminPanelPage.tsx)
   └─ Verifica user?.role === 'admin'
   └─ Redireciona para '/' se não autorizado
   └─ Usa token JWT do localStorage

2. API REQUEST
   └─ Headers: Authorization: Bearer ${token}
   └─ Realizado com axios
   └─ Para: /api/admin/dashboard

3. BACKEND (Middleware)
   └─ verifyToken() - Valida JWT
   └─ verifyAdmin() - Verifica role === 'admin'
   └─ Retorna 401/403 se não autorizado

RESULTADO: 3x de proteção = Máxima segurança ✅
